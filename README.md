# 🛒 Basket Buzz

Modern webshop built with Next.js, Tailwind CSS, and Zustand.

## 🚀 Features

- Dynamic product fetching from API
- Cart functionality with global state (Zustand)
- Responsive UI for mobile and desktop
- Clean and modern UI design

## 🛠 Tech Stack

- Next.js
- React
- Tailwind CSS
- Zustand
- REST API
